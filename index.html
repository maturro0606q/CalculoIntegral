Fórmulas esenciales del cálculo integral

Definición de Integral

La integral de una función representa el área bajo su curva o la acumulación de cantidades. Es la operación inversa de la derivada.

∫ f(x) dx = F(x) + C

Integral definida entre a y b:

∫ab f(x) dx = F(b) − F(a)

Fórmulas comunes

∫ xⁿ dx = (xⁿ⁺¹)/(n + 1) + C      (n ≠ −1)

∫ 1/x dx = ln |x| + C

∫ eˣ dx = eˣ + C

∫ sin x dx = −cos x + C

∫ cos x dx = sin x + C

∫ sec² x dx = tan x + C

∫ u dv = uv − ∫ v du      (Por partes)

Aplicaciones del Cálculo Integral

Área bajo la curva

El área bajo una curva entre los puntos a y b en el eje x se calcula como:

Área = ∫ab f(x) dx

Volumen de revolución (Método de los anillos - Eje X)

Para una función rotando alrededor del eje X:

V = π ∫ab [f(x)]² dx

Si se genera un hueco (anillo), como en funciones entre dos curvas f(x) y g(x):

V = π ∫ab ( [R(x)]² − [r(x)]² ) dx

donde R(x) es el radio externo y r(x) el interno.
